# Licensing FAQ

This FAQ is only a plain-English explanation.
The actual legal terms are in the other files and any separate written agreement for commercial use.

### What license does this plugin use?

This plugin is licensed under the PolyForm Noncommercial License 1.0.0 for public use.

That means noncommercial use is allowed under the public license.
Commercial use is not covered by the public license and requires a separate written agreement with us.

### Can I use this plugin on my private server with friends?

Yes, as long as your use is genuinely noncommercial.

### Can I modify the plugin for my own private use?

Yes, as long as your use remains noncommercial and you continue to follow the license.

### Can I share the plugin with others?

Yes, but only in ways allowed by the public license, and you must keep the license and required notices with it.

### Can I run this plugin on a public Minecraft server?

Yes, if the server use is noncommercial.
If the server is commercial or monetized, you need a separate written agreement with us first.

### What counts as commercial use?

Commercial use generally includes use connected to making money, running a business, selling services, promoting a paid offering, or supporting a commercial organization.

Examples may include (but are not limited to):
- charging players for access
- selling ranks, perks, kits, cosmetics, or subscriptions
- accepting payments tied to gameplay benefits or server access
- offering hosting or setup services that include this plugin
- using the plugin as part of a business or paid client project

### Are donations allowed?

Pure voluntary donations with no gameplay benefits, access, perks, or commercial tie-in may still raise legal questions depending on the situation.

If your server accepts donations of any kind, we strongly recommend contacting us first.
If donations are tied to rewards, access, or benefits, we will generally treat that as commercial use.

### Are paid ranks, paid cosmetics, paid perks, subscriptions, or VIP access allowed?

No, not under the public noncommercial license.
Those uses require a separate written agreement with us first.

### Can a hosting company or freelancer install this plugin for clients?

Not under the public noncommercial license if this is done as part of a paid service or commercial activity.
That requires a separate written agreement with us first.

### Can a company, organization, or server network use this plugin internally?

If the use is commercial, business-related, or tied to a monetized service, a separate written agreement with us is required.

### Can I use this plugin in videos, streams, or content creation?

Usually yes, if the plugin is just being shown incidentally while you create content.
But if the plugin itself is part of a commercial service, product, or server business, you may need a separate written agreement with us.

### Can I remove your copyright or license notices?

No. If you share the software, you must keep the required notices and license information.

### How do I discuss commercial use?

Please contact us through GitHub or via Mail `endergames.e27ft at addy.io` and include:
- your name or company name
- your server or project name
- how you want to use the plugin
- whether your server is monetized
- estimated player base or usage scope
